# langPatch
Language patcher 0.2 - 2025
The Lemon Man (C) and zSamyMan

This is a simple plugin which fakes the system language so you can
keep your xmb in english to not mess up themes and play in your
own language. Strongly recommend using it in conjunction to the
pergame plugin

To install it just copy the langPatch.prx in the seplugins folder
and activate it in the game section, then create a file called 
language.txt and insert the corresponding language code

0 - Japanese
1 - English
2 - French
3 - Spanish
4 - German
5 - Italian
6 - Dutch
7 - Portuguese
8 - Russian
9 - Korean
10 - Chinese (trad)
11 - Chinese (simp)

Note : To compile the plugin you will need the latest psp-sdk
